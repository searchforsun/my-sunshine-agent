# 考勤与加班管理规定

> 版本：v1.0  
> 生效日期：2026-06-01  
> 适用范围：全体正式员工、试用期员工

---

## 1. 目的与考勤原则

为保障正常工作秩序，明确出勤、加班与调休规则，特制定本规定。员工须按时打卡；弹性上下班核心时段为 **09:30–18:00**，可在 08:30–10:00 到岗、对应顺延下班，具体以部门排班为准。请假、调休与余额查询见《公司请假流程规范》。

---

## 2. 出勤与异常处理

| 情形 | 规则 |
|------|------|
| 迟到/早退 | 月累计 3 次记 1 次旷工半天；5 次记 1 天 |
| 忘打卡 | 3 个工作日内 OA 补签，每月不超过 3 次 |
| 外出办公 | 提前在 OA 登记外出事由、地点及预计返回时间 |
| 远程办公 | 须事前审批，按约定时段在线并响应，考勤以系统记录为准 |

考勤异常（漏打卡、设备故障等）可在 **5 个工作日内** 向直属主管及 **HR** 申诉，HR 核实后修正记录。

---

## 3. 加班管理

加班须 **提前申请**，经直属主管批准后方可计入。工作日加班优先安排调休；**周末** 加班按 1:1 换休或依法支付加班费。**法定节假日** 加班按国家规定支付三倍工资或等价调休（以劳动合同及当地法规为准）。

| 加班类型 | 调休/补偿 |
|----------|-----------|
| 工作日延长 | 1:1 调休 |
| 周末 | 1:1 调休或加班费 |
| 法定节假日 | 三倍工资或协商调休 |
| 夜班（22:00–06:00） | 夜班补贴 50 元/班 |

调休余额在 **OA/HR 系统 → 假期余额** 模块查询，与请假流程联动。路径：登录 OA → 自助服务 → 假期余额 → 查看「调休」剩余小时数。未申请或未获批的加班不计入调休；员工不得自行在系统中补录加班时长。

---

## 4. 弹性与特殊安排

核心工作时段内须保持可联系状态。因客户拜访、项目上线等需 **外出办公** 的，须提前在 OA 填写外出单并获主管批准，否则按缺勤处理。经批准的 **远程办公** 须每日在线不少于 8 小时，参加部门例会；详细申请、签到规则见《远程办公与混合办公管理办法》，**远程办公不计入请假**。长期远程须部门负责人 + HR 备案。

---

## 5. 职责分工

| 角色 | 职责 |
|------|------|
| 员工 | 按时打卡、及时补签、加班事前申请 |
| 直属主管 | 审批加班与外出、核实考勤异常 |
| HR | 制度解释、申诉受理、月度考勤汇总 |
| 部门负责人 | 长期远程、跨月调休等特殊安排 |

月度考勤结果于次月 5 日前公示，员工须在 3 个工作日内确认；有异议按申诉流程处理。

---

## 6. 常见问题

**Q：迟到几次算旷工？**  
A：月累计迟到或早退 3 次折合旷工半天，5 次折合 1 天，并影响当月绩效。

**Q：加班需要提前申请吗？**  
A：需要。须至少提前 1 个工作日提交加班申请，紧急上线等情形事后 24 小时内补录并说明。

**Q：忘打卡怎么补签？**  
A：在 OA 考勤模块申请补签，注明原因，3 个工作日内完成，每月不超过 3 次。

**Q：加班餐补标准是多少？**  
A：经批准的 **工作日晚上加班**（连续 ≥2 小时）可申领 **加班餐补 30 元/次**，随当月考勤汇总在 OA 报销，无需发票；**周末/法定节假日加班** 不另发餐补（已按加班规定补偿）。报销科目见《公司报销管理制度》，每人每月不超过 **8 次**。
