# 公司请假流程规范

> 版本：v1.0  
> 生效日期：2026-06-13  
> 适用范围：全体正式员工、试用期员工（按劳动合同及公司制度执行）

---

## 1. 目的与适用范围

为规范员工请假行为，保障部门正常运转与员工合法权益，特制定本流程。

**适用情形包括但不限于：**

| 类型 | 说明 |
|------|------|
| 年假 | 按工龄及公司制度享有的带薪年休假 |
| 事假 | 因个人事务需离岗，一般无薪 |
| 病假 | 因疾病无法正常工作，需提交证明材料 |
| 婚假 | 依法享有的带薪婚假 |
| 产假/陪产假 | 按国家及地方法规执行 |
| 丧假 | 直系亲属丧事，按公司制度执行 |
| 调休 | 加班换休，以系统/OA 中可调休余额为准 |
| 其他 | 工伤假、育儿假等按法规及公司补充制度 |

## 2. 职责分工

| 角色 | 职责 |
|------|------|
| 员工 | 提前申请、如实填写、按时销假、补充材料 |
| 直属主管 | 审批业务合理性、安排工作交接 |
| 部门负责人 | 审批跨团队/长期请假（通常 ≥3 天） |
| HR | 制度解释、假期余额核对、归档备案 |
| 行政/IT（如需） | 门禁、账号、办公设备等相关配合 |

---

## 3. 请假原则

1. **提前申请**：除突发疾病等不可抗力外，应提前提交申请。  
2. **先审批后离岗**：未经批准不得擅自离岗，否则按旷工处理。  
3. **如实申报**：病假、婚假等需按规定提供证明。  
4. **工作交接**：离岗前与主管确认交接人及待办事项。  
5. **按时销假**：返岗后及时在系统中完成销假。

## 4. 各类假期说明（参考）

> 以下为通用参考，实际以公司《员工手册》及当地法规为准。

| 假期类型 | 常见规则（参考） | 证明材料 |
|----------|------------------|----------|
| 年假 | 工作满 1 年享有，天数按工龄递增 | 一般无需 |
| 事假 | 无薪，年度上限由公司规定 | 一般无需 |
| 病假 | 按工资发放规则执行 | 医院诊断证明、病历等 |
| 婚假 | 法定天数 + 公司福利假 | 结婚证复印件 |
| 产假 | 按生育政策 | 生育相关证明 |
| 丧假 | 通常 1–3 天 | 按公司要求 |
| 调休 | 1:1 抵扣加班时长 | 以系统余额为准 |

## 5. 审批权限（示例）

| 请假时长 | 审批人 |
|----------|--------|
| ≤ 1 天 | 直属主管 |
| 2–3 天 | 直属主管 + 部门负责人 |
| > 3 天 | 部门负责人 + HR 备案 |
| ≥ 7 天或连续病假 | 部门负责人 + HR + 分管领导（如需） |

**特殊说明：**

- 关键岗位、项目节点、值班期间请假，主管可要求调整时间或加强交接。  
- 试用期员工请假规则与正式员工一致，但需更充分的工作安排说明。

## 6. 标准流程

### 6.1 流程总览

1. 员工在 OA 系统发起请假申请  
2. 直属主管审批  
3. 如需，部门负责人及 HR 审批备案  
4. 审批通过后离岗并完成工作交接  
5. 返岗后销假

### 6.2 详细步骤

#### 步骤 1：提交申请

员工在 **OA / 钉钉 / 飞书 / HR 系统** 中填写：

- 请假类型  
- 开始时间、结束时间（精确到半天/小时，按系统规则）  
- 请假天数/小时数  
- 请假事由  
- 工作交接人及交接说明  
- 紧急联系方式  

**建议提前量：**

| 请假时长 | 建议提前时间 |
|----------|--------------|
| 1 天以内 | 至少提前 1 个工作日 |
| 2–3 天 | 至少提前 3 个工作日 |
| 3 天以上 | 至少提前 1 周 |
| 年假集中使用 | 按部门排期提前沟通 |

#### 步骤 2：主管审批

主管在 **2 个工作日内**（紧急病假除外）完成审批，重点评估：

- 岗位是否有人接替  
- 是否影响项目/客户交付  
- 假期类型与余额是否匹配  

#### 步骤 3：HR 备案

审批通过后，HR 或系统自动：

- 扣减对应假期余额  
- 更新考勤记录  
- 归档审批单及附件  

#### 步骤 4：离岗与交接

- 同步团队日历/群公告（如部门要求）  
- 设置邮件/IM 自动回复（长期请假建议）  
- 完成待办交接并告知协作方  

#### 步骤 5：销假

- 按时返岗后在系统中点击 **销假**  
- 实际请假与申请不一致时，需修改并重新确认  
- 超期未归且无续假审批，按旷工或事假处理

## 7. 突发情况处理

### 7.1 突发疾病

1. 无法到岗时，**当天上班前** 通知直属主管（电话/IM）。  
2. 24 小时内补提病假申请。  
3. 返岗后 **3 个工作日内** 补交病假证明（具体时限以公司制度为准）。

### 7.2 紧急情况延长假期

1. 在原申请到期前联系主管说明情况。  
2. 提交续假申请，获批后方可继续离岗。  
3. 未续假擅自延长，超出部分按旷工或事假处理。

## 8. 违规处理（参考）

| 情形 | 处理方式 |
|------|----------|
| 未请假擅自离岗 | 按旷工处理，影响绩效考核 |
| 虚假请假、伪造证明 | 按严重违纪处理，可解除劳动合同 |
| 逾期未销假 | 考勤异常，需 HR 核实后修正 |
| 频繁临时请假影响工作 | 主管约谈，纳入绩效考量 |

## 10. 常见问题 FAQ

**Q1：半天假怎么请？**  
A：在系统中选择「上午/下午」或按小时填写，以系统配置为准。

**Q2：年假可以跨年吗？**  
A：一般当年有效，未休完是否可结转以《员工手册》为准，通常需在 Q1 前休完。

**Q3：请假期间可以处理工作吗？**  
A：可以自愿处理，但不改变请假性质；病假期间应以休息为主。

**Q4：远程办公算请假吗？**  
A：不算。远程办公需单独走远程/灵活办公审批流程。

**Q5：审批被驳回怎么办？**  
A：与主管沟通调整时间或补充说明后重新提交。
