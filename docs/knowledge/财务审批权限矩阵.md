# 财务审批权限矩阵

> 版本：v1.0  
> 生效日期：2026-06-01  
> 适用范围：费用报销、采购付款、差旅及预算相关审批

---

## 1. 目的与适用范围

为明确各级管理人员财务审批 **额度** 与职责，防范越权审批，特制定本矩阵。适用于 OA 报销单、采购付款申请、差旅审批及 **预算外** 支出。与《公司报销管理制度》《部门预算管理办法》配套使用；待审批单据列表可通过 **finance-list** workflow 查询。

---

## 2. 费用报销审批矩阵

| 金额（人民币） | 直属主管 | 部门经理 | 财务复核 | CFO |
|----------------|:--------:|:--------:|:--------:|:---:|
| ≤ 2000 | ✓ | — | ✓ | — |
| 2001–5000 | ✓ | ✓ | ✓ | — |
| 5001–20000 | ✓ | ✓ | ✓ | ✓ |
| > 20000 | ✓ | ✓ | ✓ | ✓ + 总经理 |

**部门经理** 单笔审批 **额度上限** 为 **5000 元**（不含已授权差旅包干）。超过 **5000 元** 须 **CFO** 审批；超过 20000 元须总经理联签。

---

## 3. 差旅费审批权限分级

| 差旅类型 | 审批人 |
|----------|--------|
| 市内短途 | 直属主管 |
| 国内出差（≤3 天） | 直属主管 + 部门经理 |
| 国内出差（>3 天） | 部门经理 + 财务复核 |
| 国际/港澳台 | 部门经理 + CFO |

住宿、交通上限见《公司报销管理制度》。超标须事前特批。

---

## 4. 采购付款审批矩阵

| 采购金额 | 审批链 |
|----------|--------|
| ≤ 1 万元 | 直属主管 → 财务复核 |
| 1–10 万元 | 部门经理 → 财务复核 → 财务总监 |
| 10–50 万元 | 部门经理 → 采购部 → CFO |
| > 50 万元 | 采购委员会 + 总经理 |

**预算内采购** 流程见《部门预算管理办法》；**预算外** 支出须部门负责人书面说明 + 财务总监批准，必要时报 CFO。**财务复核** 职责：审核发票合规、科目正确、附件齐全、审批链完整；不合规单据退回补正。合规性批量分析可使用 **finance-smart** workflow。采购合同须与预算科目一致，无预算余额的单据系统自动拦截。

---

## 5. 预算外与特批

| 情形 | 批准人 |
|------|--------|
| 预算内调剂（部门内） | 部门负责人 |
| 跨部门调剂 | 财务总监 |
| 预算外 ≤ 5 万 | 部门负责人 + 财务总监 |
| 预算外 > 5 万 | 部门负责人 + 财务总监 + CFO |

特批须附书面说明、影响评估及后续纠偏计划，存档备查。

---

## 6. 职责说明

| 角色 | 核心职责 |
|------|----------|
| 直属主管 | 业务真实性、必要性初审 |
| 部门经理 | 本部门预算内支出审批 |
| 财务复核 | 票据、税率、科目、流程合规 |
| CFO | 5000 元以上及重大预算外支出 |
| 出纳 | 审批完成后付款，无审批权 |

矩阵与 OA 工作流实时同步；岗位变动时 HR 须在 3 个工作日内更新审批人，避免单据悬空。历史已批单据不受矩阵变更追溯影响。

---

## 7. 常见问题

**Q：CFO 审批什么金额以上的单据？**  
A：单笔报销超过 5000 元、采购 10 万元以上、国际差旅及重大预算外支出均须 CFO 审批。

**Q：财务复核的职责是什么？**  
A：负责票据真伪、税率、科目、附件及审批链完整性审核，有权退回不合规单据。

**Q：差旅费审批权限分级怎么理解？**  
A：按出差天数与目的地分级，国内短途由主管批，长期及国际出差须部门经理及以上加签，详见上文矩阵。
