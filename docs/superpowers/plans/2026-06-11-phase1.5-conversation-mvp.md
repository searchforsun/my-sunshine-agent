# 阶段 1.5 会话 MVP — 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 补齐会话 MVP：MySQL 持久化、多轮上下文、x-user-id 隔离、流式断点续传（Stop/刷新后继续生成）。

**Architecture:** Orchestrator 统一数据面（JPA + Flyway + ConversationService）；BFF 纯转发；前端 chatStore 改后端 API 为主存储。建议顺序 A → B → C → D（含 F 联调）→ E。

**Tech Stack:** JDK 21, Spring Boot 3.2.9, JPA + Flyway + MySQL, WebFlux SSE, Vue3 + Pinia

**设计 spec：** [2026-06-11-phase1.5-conversation-mvp-design.md](../specs/2026-06-11-phase1.5-conversation-mvp-design.md)
