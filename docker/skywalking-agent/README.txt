SkyWalking Java Agent 下载说明
===============================
版本: 9.7.0（与服务器 8.140.48.6 一致）

下载地址:
  https://skywalking.apache.org/downloads/

直接下载:
  curl -L https://archive.apache.org/dist/skywalking/java-agent/9.7.0/apache-skywalking-java-agent-9.7.0.tgz | tar -xz
  cp skywalking-agent/skywalking-agent.jar ./

放置位置:
  将 skywalking-agent.jar 放到此目录下
