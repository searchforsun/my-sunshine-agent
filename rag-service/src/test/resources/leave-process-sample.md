# 公司请假流程规范

> 版本：v1.0  
> 生效日期：2026-06-13  
> 适用范围：全体正式员工、试用期员工（按劳动合同及公司制度执行）

---

## 1. 目的与适用范围

为规范员工请假行为，保障部门正常运转与员工合法权益，特制定本流程。

**适用情形包括但不限于：**

| 类型 | 说明 |
|------|------|
| 年假 | 按工龄及公司制度享有的带薪年休假 |
| 事假 | 因个人事务需离岗，一般无薪 |
| 病假 | 因疾病无法正常工作，需提交证明材料 |
| 婚假 | 依法享有的带薪婚假 |
| 产假/陪产假 | 按国家及地方法规执行 |
| 丧假 | 直系亲属丧事，按公司制度执行 |
| 调休 | 加班换休，以系统/OA 中可调休余额为准 |
| 其他 | 工伤假、育儿假等按法规及公司补充制度 |

## 10. 常见问题 FAQ

**Q1：半天假怎么请？**  
A：在系统中选择「上午/下午」或按小时填写，以系统配置为准。

**Q2：年假可以跨年吗？**  
A：一般当年有效，未休完是否可结转以《员工手册》为准，通常需在 Q1 前休完。

**Q3：请假期间可以处理工作吗？**  
A：可以自愿处理，但不改变请假性质；病假期间应以休息为主。

**Q4：远程办公算请假吗？**  
A：不算。远程办公需单独走远程/灵活办公审批流程。

**Q5：审批被驳回怎么办？**  
A：与主管沟通调整时间或补充说明后重新提交。
